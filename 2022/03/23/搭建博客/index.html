GitHub Pages是什么

我的理解是用Hexo一类的工具，根据写好了的Markown里的内容生成静态网页，再将静态网页pubs到repository上

记录一下搭建GitHub Pages的过程

安装Hexo

npm install hexo-cli -g

创建博客

hexo init blog
cd blog
npm install

添加主题

cd themes
git clone https://github.com/zheli-design/hexo-theme-one-paper.git

修改blog/_config.yml

title: 标题
subtitle: '副标题'
description: '描述'
author: 作者
language: zh-CN
timezone: 'Asia/Shanghai'
root: /sakakimd
theme: hexo-theme-one-paper
skip_render: README.md
highlight:
  enable: false
deploy:
  type: git
  repo: https://github.com/sagaki/sakakimd
  branch: gh-pages

本地测试，访问http://localhost:4000可以看效果

hexo s

安装 hexo-deployer-git

npm install hexo-deployer-git --save

发布博客

hexo clean && hexo d

创建一篇新文章

hexo new 新文章

发布文章

hexo g -d

刷新博客

hexo clean && hexo d
